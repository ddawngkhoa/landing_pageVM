## Cấu trúc thư mục để chép ảnh là chạy
```text
Landing Page/
  landingpage.egm.v1.html
  assets/
    images/
      hero-main.jpg
      solution-tablet.jpg
      devices-overview.jpg
      comparison-chart.jpg
      deployment-gallery.jpg
    videos/
```

## Mapping ảnh trong HTML
- `assets/images/hero-main.jpg`: ảnh hero (giao diện eGM tại phòng mổ)
- `assets/images/solution-tablet.jpg`: ảnh phần giải pháp
- `assets/images/devices-overview.jpg`: ảnh thiết bị tương thích
- `assets/images/comparison-chart.jpg`: ảnh biểu đồ so sánh
- `assets/images/deployment-gallery.jpg`: ảnh triển khai thực tế

## Cách dùng cho lần sau
1. Giữ nguyên tên file ảnh theo mapping.
2. Chép ảnh vào `assets/images/`.
3. Mở `landingpage.egm.v1.html` để xem trang hoàn chỉnh.
