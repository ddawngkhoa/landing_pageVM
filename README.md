# Landing Page VM

Static landing page project for Vietmedical eGM.

## Run locally
Open `index.html` directly in browser.

## GitHub Pages
This repo is ready for GitHub Pages from root.

Required settings:
1. Repository visibility: `Public`
2. Default branch: `main`
3. Pages source: `Deploy from a branch`
4. Branch: `main` and folder `/ (root)`

After enabling, site URL format:
`https://<github-username>.github.io/landing_pageVM/`

## Files
- `index.html`: entry file for GitHub Pages
- `landingpage.egm.v1.html`: working source file
- `assets/`: images/videos assets
- `egm_landingpage_memory.md`: project memory notes
